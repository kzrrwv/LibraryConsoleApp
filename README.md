# LibraryConsoleApp
  Этот проект представляет собой систему управления библиотекой, реализованный на Java. Он позволяет пользователю выполнять базовые функции библиотеки(брать книгу, возвращать её, добавлять и т.д.)
### Основные функции:
    Добавить книгу - добавляет книгу в лист.
    Добавить читателя - добавляет читателя в лист.
    Взять книгу - береётся книга из добавленных.
    Вернуть книгу - возвращается книга.
    Показать все книги - выводиться содержимое листа с книгами.
    Показать всех читателей - выводится содержимое листа с читателями.
    Найти книгу - происходит поиск книги по её названию. 
    Найти читателя - происходит поиск читателя по его имени.
    Показать просроченные книги - выводяться все книги из листа с просроченными книгами.
    Выйти - система заканчивает свою работу.
### Установка и запуск
    Скачайте IntelliJ IDEA
    Откройте проект в IntelliJ IDEA, Eclipse или другой Java-совместимой IDE
    Импортируйте все файлы .java
    Запустите программу в классе Main

Библиотечная система прошла комплексное тестирование, включающее:

- Интеграционное тестирование - взаимодействие между классами Book, Reader, Library, Menu
- Тестирование производительности - оценка времени выполнения библиотечных операций
- Юзабилити-тестирование - удобство использования консольного интерфейса для библиотекарей

## Результаты тестирования
